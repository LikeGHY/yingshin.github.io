fill函数给定区间[fist, last)的区间的元素赋同一个值。

###function template

```
template<class ForwardIt, class T>
void fill(ForwardIt first, ForwardIt last, const T& value);
```


<!--more-->


###函数说明
对[first, last)区间统一赋值。

```
template<class ForwardIt, class T>
void fill(ForwardIt first, ForwardIt last, const T& value)
{
    for (; first != last; ++first) {
        *first = value;
    }
}
```


###code

```
int main()
{
    int a[10] = {1};
    std::fill(a, a + 10, 100);
    for (int i = 0; i < 10; ++i) {
        std::cout << a[i] << std::endl;
    }

    return 0;
}
```
